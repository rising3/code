## Requirements

- Major Linux distributions that support systemd.
- awk
- vscode

## How to install

To install:

``` bash
curl -s "https://rising3.github.io/code/get/" | bash
```
## How to build from source

```sh
git clone https://github.com/rising3/code.git
cd code
npm i
npm run test
npm run build
```
