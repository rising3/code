Installation

Go on then, paste and run the following in a terminal:

curl -s "https://rising3.github.io/code/get/" | bash

Uninstallation

Go on then, paste and run the following in a terminal:

systemctl --user stop code
systemctl --user disable code
rm ~/.config/systemd/usr/code*
